Alert massage

Create a basic "Hello World" script - the simplest, most basic script that you can create in JavaScript, or any programming language.

  1. Create a basic, empty web page using only html code. The page should be completely empty except for the doctype, html, head, title, and body tags that are necessary to make a valid html document.
  2. Insert a set of script tags between the body tags.
  3. Add alert("Hello World!") between the script tags.

Final result: "Hello World!" should immediately display in an alert message when the page loads.

Variables

  • Collect information from the user and store in a variable for later user.

Final result: When the page loads, the user is prompted for his or her name. That name is then immediately displayed as Hello username! in an alert message, and then should immediately display as Hello username! in the page.

Concatenation

This assignment is an introduction to concatenation, an invaluable operation that allows you to connect together pieces of JavaScript code

  1. Create a basic page in html.
  2. Using document.write(), display the user's browser information, screen height and width in the page.
  3. The script can only use one document.write() statement, so you must use the concatenation operator + to display the information

Final result: "Your browser is [browser info here]. Your screen height is [screen height here]. Your screen width is [screen width here]."
